# 0verMalaysiaSpring
馬春班

## Contributing
### Install Dependencies
```
git clone https://github.com/0verseas/0verMalaysiaSpring.git
cd 0verMalaysiaSpring
npm install
```
### Setup
```
cp src/env.js.example src/env.js
```
edit the config file in `src/env.js`

### Testing
```
$ npm run serve
```

### Building
```
$ npm run build
```
